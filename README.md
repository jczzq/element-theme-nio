## Installation

## Usage

Use Sass import
```css
@import 'element-theme-nio';
```

Or Use webpack
```javascript
import 'element-theme-nio';
```

Or
```html
<link rel="stylesheet" href="path/to/node_modules/element-theme-nio/lib/index.css">
```

##  Import on demand
```javascript
import 'element-theme-nio/lib/input.css';
import 'element-theme-nio/lib/select.css';

// ...
```
